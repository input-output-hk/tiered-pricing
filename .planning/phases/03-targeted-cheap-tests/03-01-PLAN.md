    1. `pub struct CiResult { pub point: f64, pub lower: f64, pub upper: f64, pub alpha: f64, pub n_bootstrap: u32, pub bootstrap_seed: u64 }` with `#[derive(Debug, Clone, Copy, PartialEq, serde::Serialize, serde::Deserialize)]`.     2. `pub struct DeltaSummary { pub n: usize, pub median: f64, pub iqr: f64, pub sign_coherence: f64 }` with the same derives. `sign_coherence` is the fraction of paired deltas (`samples_a[i] - samples_b[i]`) sharing the sign of the median delta, in `[0.0, 1.0]`.

    3. `pub fn paired_bca_ci(samples_a: &[f64], samples_b: &[f64], alpha: f64, bootstrap_seed: u64) -> CiResult` — implements paired-sample BCa (Bias-corrected and accelerated) bootstrap confidence intervals on the mean of `delta[i] = samples_a[i] - samples_b[i]`. Uses `rand_chacha::ChaCha20Rng::seed_from_u64(bootstrap_seed)` for resampling (seeded same way the simulator seeds its RNGs per CLAUDE.md determinism contract). Bootstrap iterations: `n_bootstrap = 9999` (Claude's Discretion default per CONTEXT.md). Uses `statrs::distribution::{Normal, ContinuousCDF}` for the BCa z-score → percentile transform. Panics with a clear message if `samples_a.len() != samples_b.len()` (paired requires equal lengths) or if either is empty or if any sample is not `f64::is_finite()`. `alpha` is the two-sided confidence level (e.g. 0.05 for a 95% CI); panic if not in `(0.0, 1.0)`.

    4. `pub fn paired_delta_summary(samples_a: &[f64], samples_b: &[f64]) -> DeltaSummary` — computes median, inter-quartile range (IQR), and sign-coherence of the paired deltas. Uses linear-interpolation percentile via a sorted copy of the deltas; q25 = percentile(0.25), q75 = percentile(0.75), IQR = q75 - q25. `sign_coherence` rounds toward zero so a 0-delta sample neither agrees nor disagrees (treat 0.0 as agreeing with the median's sign for stability; document choice in doc-comment). Panics under the same input guards as `paired_bca_ci`.
